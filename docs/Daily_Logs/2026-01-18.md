Today we added a **non-intrusive visual overlay** built with **Tauri v2** that behaves exactly like Apple's Siri overlay.

### What Was Built

1. **Tauri v2 Overlay** (`gui-overlay/`)
   - Rust backend with click-through window configuration
   - HTML/CSS frontend with animated orb and waveform
   - HTTP-based IPC for Python communication

2. **Python Controller** (`core/overlay.py`)
   - HTTP server on port 9877 for state communication
   - Simple API: `overlay.listening()`, `overlay.thinking()`, etc.
   - Automatic lifecycle management

3. **Main Loop Integration** (`main.py`)
   - State changes at each step of the voice interaction
   - Error state handling

### Key Technical Details

The overlay achieves non-interruption through:

```rust
window.set_ignore_cursor_events(true);
```

Combined with these Tauri window settings:
- `alwaysOnTop: true`
- `decorations: false`
- `transparent: true`
- `focus: false`
- `skipTaskbar: true`
